# INVENTORY-CONTROL--SYSTEM
This project  helps students understand how business track products ,stock levels,and inventory movement using Excel. Students will manage Inventory data to monitor available stock ,identify low stock items and calculate inventory value.

## **Here are some main skills one will acquire through out this project:**
1. Organize Inventory data in Excel tables
2. Track stock in and stock out transanctions
3. Calculate current stock out transanctions
4. Apply conditional formulas
5. Identify low stock and out of stock items
6. Calculate inventory value
7. Create inventory summary reports and charts
